# Esponenziale

Programma Java che calcola la potenza di un numero intero. L'utente inserisce base ed esponente e il programma restituisce il risultato calcolato tramite un ciclo iterativo, senza utilizzare librerie matematiche esterne.

## Come funziona

Il calcolo viene eseguito moltiplicando la base per se stessa tante volte quante indica l'esponente, implementando manualmente l'operazione di potenza con un ciclo `while`.

## Esempio di output

```
Inserisci la base: 2
Inserisci l'esponente: 4
Risultato: 16
```

```
Inserisci la base: 3
Inserisci l'esponente: 5
Risultato: 243
```

## Limitazioni

- Accetta solo numeri interi
- L'esponente deve essere maggiore o uguale a 1

## Tecnologie

- Java

## Come eseguire

```bash
javac esponenziale.java
java esponenziale
```
